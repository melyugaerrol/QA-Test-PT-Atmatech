top_location
	Displays location that are often discussed and also total of the location in the topic.
	If we click one of the 'location', will display twitter post that writes the location 
	There is a filter to select Twitter posts that contain the selected 'location' based on :
	1. Recent tweets
	2. Oldest tweets
	3. Popular tweets
	4. Most liked tweets
	5. Most replied tweets
	6. Most retweeted
